COVID-19 Hackathon / Containment measures effectiveness / Luca Ferrari¶

Please consider that the following analysis has been conducted with data updated to April 3, 2020 .

Problem Definition / Objective¶

COVID-19 is spreading around the world at a fast pace.
Data analysis is required to get a sense of the pandemic and assess the effectiveness of the containment measures.
This analysis provide a new way of visualizing the growth data and tries to estimate how the different italian goverment containment measures impacted on the actual virus growth in Italy.

How do we know when we reach the inflection point?

Some further processing and analysis is required.

As I read in different articles (one of them being: https://shorturl.at/ewV78) one important number to compute is R0 which is the average number of secondary infection and if that is < 1 it means that the epidemic can be contained.

R0 can be computed by $\frac{new cases}{total cases}$, so we need to calculate the number of new cases.

1. How do I recognize that exponential growth is ending?¶

If the line starts pointing downwards when a certain total number of cases is reached it means that R0 is getting lower and so there will be less and less new cases which obviously means that the exponential growth is ending (sigmoid curve flattening).

We can easily see it if we plot the chinese data.
As we all know, China has already finished its quarantine, as reported by medias, and so we can se the effect of that in the growth of the number of the new cases.

2. Do the growth rates of the number of cases differ between nations?¶

To verify if the growth rates of the number of cases differ between nations we need to evaluate the different R0 and then compare them.
Since this report part is all about visualization and inferring new information just by looking at the data, we can simply plot the different growth rates and esitimate the differences without actually computing anything.
Similar growth rates will have approximately the same slope so we can verify that very quickly.

To reduce the variation we can apply the same technique as before and filter the new cases and the cnt date with a moving average.

Comments¶

What we expected to find was:

  • A large positive coefficient for log_cnt since, as we said earlier, there's a linear relationship between that and log_new_cases. This makes log_cnt a strong predictor and we can assess it by looking at the p-value which is 0.00.
  • Negative coefficients for all the containment measures features. We obviously would expect a negative growth of the number of cases if quarantine measures are applied. However, what we actually got is a statistically significant negative coefficient just related to social distancing (p-value = 0.00). Moreover positive and almost statistically significant coeffiecients values were estimated for s_schools and s_commercial_act. Nevertheless those coefficients were almost 0 so they can't be taken into account as predictors.

Potential problems:

  • We need to take into account that some measures are less strict then others and so the effect of these measures is delayed in time. This delay can't be properly detected by the model and it confuses measures with each others.

  • Multicollinearity

  • Lack of data of negative growth. It is possible that the model can't properly predict the falling of the number of new cases because we are in the beginning of the descent (Italy).

Let's try to manage the delay of the effect of the containment measures and re-train the model.

Comments¶

As we can see almost all of the coefficients are now negative as we expected (except log_cnt which is still the main postive predictor), although only two of them are statistically signficant.

The result of the analysis is closer to what we expected, however is possible that the result is distorted because of the wrong assumptions on the delayed effects of the quarantine measures, because of multicollinearity of data or as we said before, because of a lack of data on the descent phase.

The large value of R2 is probably due to the fact that basically all of the variability is explained by the log_cnt predictor.

Conclusions¶

We saw that it is possible to assess the containment measures efficacy looking at the number of new cases against the number of total cases and we discovered that the virus basically spreads equally among all the different nations taken into account. It is very likely that it is possible to extend what we found to the entire world nations, however a more rigorous analysis has to be carried out.

More data is needed and further analysis have to be done in order to understand how the single different quarantine measures impact on the number of new cases, giving us the possibility to predict it and develop strategies to contain the epidemic.
